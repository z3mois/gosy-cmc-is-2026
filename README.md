# Конспект к госэкзамену

Единый LaTeX-конспект по билетам к государственному экзамену магистерской программы «Интеллектуальные системы».

## Структура

- `gosy.tex` — главный файл документа.
- `tickets/ticketXX.tex` — отдельные билеты.
- `gosy.pdf` — собранная актуальная версия конспекта.
- `AGENTS.md` — правила редактирования материалов.

## Сборка

Для сборки нужен установленный LaTeX с поддержкой `latexmk`.

```powershell
latexmk -pdf -interaction=nonstopmode gosy.tex
```

Служебные файлы сборки игнорируются через `.gitignore`.
